## 🎯 What is ResumeIQ Pro?

ResumeIQ Pro is a *professional-grade AI Resume Analyzer* that helps job seekers optimize their resumes for any job description. It uses *Claude AI* to provide real-time ATS scoring, keyword analysis, and actionable suggestions — just like having a personal career coach.

---

## ✨ Features

| Feature | Description |
|--------|-------------|
| 🎯 *ATS Score* | Animated ring visualization with real-time scoring |
| 🧠 *NLP Keyword Extraction* | Identifies matching and missing keywords |
| 📊 *Score Breakdown* | 5-category analysis: Keywords, Experience, Skills, Formatting, Education |
| 🎭 *Role-Specific Analysis* | Tailored for 10+ roles including Data Analyst, ML Engineer, SWE |
| 💡 *Smart Suggestions* | Priority-based actionable recommendations |
| 📄 *PDF Upload* | Analyze actual resume files |
| 📥 *Download Report* | Export full analysis as text report |
| 🔄 *Multiple Resume Comparison* | Compare multiple resumes side by side |

## 🖥️ Tech Stack

*Frontend*
- React 18 + Vite
- CSS-in-JS (no external UI library)
- Claude AI API (Anthropic)

*Backend / Deployment*
- Python + Streamlit
- PyMuPDF for PDF parsing
- Deployed on Vercel + Streamlit Cloud

## 🚀 Getting Started

### Prerequisites
- Node.js 18+
- Claude API Key from [console.anthropic.com](https://console.anthropic.com)

### Installation

```bash
# Clone the repository
git clone https://github.com/kiru-builds/resumeiq-pro.git

# Navigate to project
cd resumeiq-pro

# Install dependencies
npm install

# Start development server
npm run dev
